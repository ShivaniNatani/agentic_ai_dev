# Convergence Launch — IKS Exclusive (Week of Jan 30, 2026)

**TL;DR:** Convergence is our new single surface that blends the dashboard, agent console, and reporting lens into one flow. This first release is available **only on IKS** and is ready for this week’s stakeholder newsletter.

## Email-Ready Copy
**Subject:** [IKS Exclusive] Convergence is live — dashboard + agents + reporting in one place

Hi team,

We’re introducing **Convergence**, the unified experience for IKS that fuses our operational dashboard, agent workflows, and reporting into a single, always-synced view. It’s designed to cut swivel-chair time, keep decisions in context, and get exec-ready narratives without leaving the workspace.

**What’s inside now**
- **Unified Dashboard:** Cross-domain health (clinical ops, RCM, platform) with real-time signals and instant state sync across widgets.
- **Agent Console:** Pre-wired agent lanes (Prior Auth, Claims, Referral) plus orchestration controls and audit visibility so handoffs stay traceable.
- **Narrative Reporting:** Auto-generated briefs that translate live metrics into actions; tuned for weekly governance and daily standups.

**Availability**
- Convergence is **exclusive to the IKS environment** in this first wave. Access with your existing IKS SSO/LDAP credentials.
- Best on Chrome or Edge (desktop); mobile view is supported for read-only monitoring.

**How to engage this week**
- Try the Convergence tab in the dashboard and trigger an agent workflow end-to-end.
- Share one insight from the reporting view with your team; reply with blockers or data gaps.

Thanks for championing this launch—your feedback this week will shape the broader rollout.
