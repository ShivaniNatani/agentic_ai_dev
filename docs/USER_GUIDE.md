## 🔐 Logging In

### Step 1: Enter Credentials
1. Navigate to the login page
2. Enter your username and password
3. Click **"Continue →"**

### Step 2: MFA Verification
1. A second screen will appear asking for a 6-digit code
2. Enter your MFA code (demo: `123456`)
3. Click **"Verify & Sign In"**

### Quick Demo Access
Use the quick buttons at the bottom of the login page:
- **Admin**: Full access to all features
- **Developer**: Development tools and APIs
- **User**: Read-only dashboard access

## 🎯 Optimix Healthcare

The Optimix dashboard provides specialized insights for healthcare revenue cycle management (RCM) and claims processing.

### IKS Insights Dashboard
Navigate to **Dashboard > Optimix > IKS Claims** to access:
- **Projected vs Actuals**: Compare prediction models against real-world payment and denial outcomes.
- **Workable Inventory**: Monitor the backlog of actionable claims.
- **Calendar Drilldown**: Click any date to view granular daily metrics.

### 🤖 Orbit AI Assistant
Activate the Orbit assistant via the **"Orbit AI"** button in the header. Ask questions such as:
- *"What is the denial trend for last week?"*
- *"Explain the workable formula components."*
- *"Show me the breakdown for August 15th."*

### 📅 Calendar Features
- **Heatmap Visualization**: Days are color-coded by volume intensity.
- **Details Panel**: Select a day or month view to see raw data tables.

## ❓ FAQ

### How do I switch between Agentic and MLOps views?
On the main dashboard, use the toggle buttons labeled "Agentic Dashboard" and "MLOps Observatory".

### I forgot my password, what do I do?
Contact your administrator or use the demo credentials listed in the README.

### Why do I need to enter an MFA code every time?
MFA (Multi-Factor Authentication) is required for security. Use code `123456` for demo access.

### How do I access a specific agent?
Navigate to AI Agents → Click on the agent card → View details and documentation.

### Who do I contact for support?
- Email: support@ikshealth.com
- Website: [ikshealth.com/contact-us](https://ikshealth.com/contact-us/)
