Detailed Changelog

Comprehensive breakdown of all system updates.

Optimix (Operational Intelligence)
  • New Dashboard Module: Dedicated accessible tab for GIA & AXIA clients.
  • Financial Metrics: Real-time tracking of Total Open Claims, $AR, and Cash Collected.
  • Inventory Analytics: Burn Rate visualization, Workable vs. Backlog breakdown.
  • Google Sheets Backend: Direct integration with private Sheets via Spreadsheet ID.
  • Mock Data Fallback: Automatic fallback to simulated data if API connection fails.
  • Date filtering: Daily activity tracking with trend line analysis.
Agentic AI (Swarm Ops)
  • Agent Catalog: Centralized library of all active agents (Coder, Researcher, Verifier).
  • Interactive Sandbox: In-browser chat interface to test agent capabilities safely.
  • Neural Trace v2.0: Fixed "sticky" scrolling issues for smoother reasoning trail debugging.
  • Responsive Layout: Improved mobile compatibility for trace logs and agent details.
  • Status Indicators: Real-time "Live" / "Offline" status for each agent.
MLOps Observatory (Model Health)
  • System Health Monitor: Green/Red status dashboard for API endpoints.
  • Drift Detection: Automated alerts when model outputs diverge >5% from baseline.
  • Light Mode Fixes: complete UI overhaul for MLOps cards to support the new "Cloudy Blue" theme.
  • Performance Metrics: Latency charts (P95, P99) and Error Rate tracking.
Platform Infrastructure
  • Production Server: Migrated from Flask Dev to Gunicorn (4 workers, 2 threads).
  • Security Hardening: API Key auth for Webhooks, disabled debug routes.
  • Unified Auth: LDAP integration with "Cloudy Blue" login page & RBAC support.
  • Dynamic Release Notes: Updates feed powered by Google Sheets (no code deploy needed).
  • Docker Optimization: Simplified run command (Port 8510 default) + Non-root user execution.
  • UI/UX Overhaul: New "Cloudy Blue" light theme with high-contrast palette.

Frequent Questions

Launch readiness info.

How often do models retrain?
Models are evaluated every 6 hours. If drift exceeds 0.05, retraining triggers automatically in the sandbox.
Is patient data secure?
Yes. All data undergoes automated de-identification at the ingestion layer (Layer 1).
What if an agent fails?
The swarm controller automatically respawns the agent. If failure persists, it routes to human review.
Can I export reports?
Yes. Navigate to 'System Health' in the console and click 'Export CSV' to get metrics.
